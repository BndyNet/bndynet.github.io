## Links

```markdown
[text](url)           //<a href="url">text</a>
![alt text](image_url)    //<img src="image_url" alt="alt text" />
```

## Table

```markdown
| Tables        | Are           | Cool  |
| ------------- |:-------------:| -----:|
| col 3 is      | right-aligned | $1600 |
| col 2 is      | centered      |   $12 |
| zebra stripes | are neat      |    $1 |
```
